如何清除 Git 仓库的所有提交记录

Authors
  • Name
    Later
    Twitter
  1. 切换到新的分支

git checkout --orphan latest_branch

  1. 缓存所有文件

git add -A

  1. 提交跟踪过的文件

git commit -am "commit message"

  1. 删除 master 分支

git branch -D master

  1. 重命名当前分支为

master git branch -m master

  1. 提交到远程 master 分支

git push -f origin master
